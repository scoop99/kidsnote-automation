# 마이 알림장 뷰어 (KAV) & 자동 백업

키즈노트에 업로드된 소중한 우리 아이의 추억(알림장, 앨범)을 내 컴퓨터로 안전하게 백업하고, 아름다운 전용 뷰어로 감상할 수 있는 통합 도구입니다.

---

## ✨ v1.3.0 주요 업데이트 (릴리즈 노트)

이번 버전에서는 백업된 데이터를 더욱 가치 있게 활용할 수 있는 **KAV 전용 뷰어**가 정식 탑재되었습니다.

*   🖥️ **전용 웹 뷰어(KAV) 출시**: 단순 파일 보관을 넘어, 실제 키즈노트 앱처럼 세련된 UI로 사진과 글을 감상하세요. (다크모드 지원)
*   🧠 **스마트 동기화 고도화**: 과거에 이미 백업된 데이터도 분석하여 누락된 상세 정보(metadata.json)를 자동으로 복구하고 최신 상태로 유지합니다.
*   ⏰ **한국 표준시(KST) 완전 지원**: 댓글 작성 시간 등이 세계 표준시가 아닌 한국 시간으로 정확하게 표시됩니다.
*   📊 **생활기록 추출 기능**: 아이의 기분, 건강 상태, 체온, 식사량 등 상세 알림장 데이터를 자동으로 추출하여 뷰어에서 보여줍니다.
*   ⚡ **성능 안정화**: 대규모 백업 시의 속도 최적화 및 비정상 종료 방지 로직이 강화되었습니다.

---

## 🚀 사용 순서 (이것만 기억하세요!)

폴더 안에 있는 **`[여기를_더블클릭하세요].bat`** 파일을 실행하면 모든 백업 관리를 편하게 하실 수 있습니다.

### 1단계: 프로그램 준비 (처음 한 번만)
화면에 뜨는 메뉴 중 **1번(프로그램 설치)**을 눌러 필요한 준비를 완료하세요.

### 2단계: 로그인 및 백업 시작
메뉴에서 **2번(즉시 백업 시작)**을 눌러 백업을 시작합니다.
- 처음 1회 실행 시 **인터넷 창이 뜨면 키즈노트 로그인을 직접 완료**해 주세요.
- **이미 받은 자료는 0.1초 만에 건너뛰고**, 새로운 글이나 댓글이 달린 글만 골라 빠르게 저장합니다.

### 3단계: 내 알림장 감상하기 (KAV 뷰어)
메뉴에서 **5번(내 알림장 보기)**을 누르면 브라우저가 열리며 백업된 내용을 감상할 수 있습니다.
- 날짜별/월별 필터링, 검색 기능을 지원합니다.
- 슬라이드 쇼 형태의 사진 크게 보기(라이트박스)를 지원합니다.

### 4단계: 매일 자동으로 저장하기 (예약)
메뉴에서 **3번(백업 시간 예약)**을 누르고 원하는 시간을 입력하세요.

### 5단계: 저장 경로 바꾸기
메뉴에서 **7번(경로 변경)**을 눌러 사진이 저장될 위치를 바꿀 수 있습니다.

## 📁 내 사진은 어디에 어떻게 저장되나요?

지정하신 저장 폴더 안에 다음과 같이 깔끔하게 정리됩니다.

*   **`앨범` / `알림장`**: `YYYY/MM` 계층 구조로 정리되어 원본 화질의 사진과 영상이 보관됩니다.
*   **`metadata.json`**: 사진 정보, 댓글, 생활기록 등이 구조화된 데이터로 저장되어 뷰어 엔진이 이를 활용합니다.
*   **`내용.txt`**: 뷰어 없이도 언제든 메모장으로 읽을 수 있는 텍스트 파일입니다.

---

## 💡 자주 묻는 질문 (FAQ)

*   **시간이 이상해요**: 모든 시간 데이터는 **한국 시간(KST)**으로 정확하게 변환되어 저장됩니다.
*   **아이콘이 안 보여요**: 뷰어에서 사용하는 아이콘은 시스템 기본 이모지입니다. 최신 윈도우 사용을 권장합니다.
*   **이름이 왜 바뀌었나요?**: 상업적 상표권 문제를 방지하고 서비스의 독립성을 확보하기 위해 **KAV(KidsNote Archive Viewer)**라는 고유 명칭을 사용하게 되었습니다.

## 🧹 프로그램 삭제 방법

1.  프로그램 실행 후 **4번 (예약 해제)** 메뉴를 눌러 백업 일정을 먼저 지웁니다.
2.  이후 프로그램 폴더 전체를 삭제하시면 완벽하게 끝납니다.
