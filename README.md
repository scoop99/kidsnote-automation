# KidsNote Automation (키즈노트 자동 백업 툴)

키즈노트에 업로드된 소중한 우리 아이의 추억(알림장, 앨범)을 내 컴퓨터로 안전하고 편하게 백업해주는 자동화 도구입니다.

## 주요 기능
- **자동 페이징**: 앨범과 알림장의 모든 기록을 자동으로 탐색하여 다운로드합니다.
- **이미지 & 영상 백업**: 원본 화질 그대로의 사진과 동영상을 날짜별로 정리합니다.
- **본문 & 댓글 저장**: 게시글의 본문 내용과 선생님이 달아주신 댓글을 텍스트 파일로 함께 보관합니다.
- **자동 업데이트**: 프로그램 실행 시마다 GitHub을 통해 최신 기능을 자동으로 유지합니다.

## 관리자 (Maintainer) 가이드
직접 코드를 수정하거나 배포하시려는 경우, 다음을 참고하세요.

1. **환경 설정**: `system/config.json`에서 저장소 이름(`github_repo`)과 기본 설정값을 확인하세요.
2. **배포**: 업데이트 시 `system/package.json`의 버전을 올리고 GitHub Release(vX.X.X 태그)를 생성하면 사용자들에게 자동 전파됩니다.
3. **보안**: `config.json`과 `session.json`은 `.gitignore`에 의해 보호되므로, 개인 정보 유출 걱정 없이 푸시하셔도 됩니다.
