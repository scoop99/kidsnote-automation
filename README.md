# KidsNote Automation (키즈노트 자동 백업 툴)

키즈노트에 업로드된 소중한 우리 아이의 추억(알림장, 앨범)을 내 컴퓨터로 안전하고 편하게 백업해주는 자동화 도구입니다.

---

## 🚀 사용 순서 (이것만 기억하세요!)

폴더 안에 있는 **`[여기를_더블클릭하세요].bat`** 파일을 실행하면 모든 백업 관리를 편하게 하실 수 있습니다.

### 1단계: 프로그램 준비 (처음 한 번만)
화면에 뜨는 6가지 파란색 메뉴판 중 **1번(설치)**을 눌러 필요한 준비를 완료하세요. (필요한 프로그램들이 자동으로 설치됩니다.)

### 2단계: 로그인 및 백업 시작
메뉴에서 **2번(백업 시작)**을 눌러 백업을 시작합니다.
- 처음 1회 실행 시 터미널(검은 창)과 함께 **인터넷 창이 뜨면 키즈노트 로그인을 직접 완료**해 주세요.
- 한 번 로그인하면 다음부터는 창이 뜨지 않고 자동으로 저장됩니다.
- **스마트 동기화**: 이미 받은 자료는 0.1초 만에 건너뛰고, **새로운 글이나 댓글이 달린 글만** 골라 아주 빠르게 저장합니다.

### 3단계: 매일 자동으로 저장하기
메뉴에서 **3번(예약)**을 누르고 원하는 시간(숫자만, 예: 18)을 입력하세요.
- 매일 지정된 시간에 컴퓨터가 알아서 백업을 수행합니다.

### 4단계: 저장 경로 바꾸기
메뉴에서 **6번(경로 변경)**을 눌러 사진이 저장될 위치를 바꿀 수 있습니다.
- 기본값 위치: 프로그램 폴더 안의 `downloads` 폴더

---

## 📁 내 사진은 어디에 어떻게 저장되나요?

지정하신 저장 폴더 안에 다음과 같이 깔끔하게 정리됩니다.

*   **`앨범` / `알림장`**: 날짜별로 구분되어 원본 화질의 사진과 영상이 보관됩니다.
*   **`내용.txt`**: 알림장 본문 글과, 매일매일 달리는 **부모님과 선생님의 댓글**까지 하나도 빠짐없이 예쁘게 저장됩니다.

---

## 💡 자주 묻는 질문 (FAQ)

*   **시간이 이상해요**: 세계 표준시가 아닌 **한국 시간(KST)**으로 정확하게 변환되어 저장됩니다.
*   **백업 속도가 너무 빨라요**: 바뀐 내용이나 새로운 댓글이 없는 게시글은 0.1초 만에 확인하고 통과하기 때문입니다. 누락이 아니니 안심하세요!
*   **컴퓨터를 옮겼는데 실행이 안돼요**: 프로그램을 다른 폴더나 다른 컴퓨터로 옮겼다면, 반드시 다시 **3번(예약)** 메뉴를 눌러 스케줄을 재등록해 주세요!

---

## 🧹 프로그램 삭제 방법

1.  프로그램 실행 후 **4번 (자동 백업 예약 해제)** 메뉴를 눌러 백업 일정을 먼저 지웁니다.
2.  이후 다운로드한 프로그램 **폴더 전체를 마우스 오른쪽 버튼으로 삭제**하시면 완벽하게 끝납니다.
