# 키즈노트 자동 백업 툴 사용 설명서

이 프로그램은 키즈노트의 알림장과 앨범 데이터를 자동으로 내 컴퓨터에 백업해주는 도구입니다.

## 🚀 시작하기

1. **압축 풀기**: 받은 압축 파일을 원하는 폴더에 풀어주세요.
2. **프로그램 실행**: 폴더 안에 있는 **`[여기를_더블클릭하세요].bat`** 파일을 더블 클릭합니다.
3. **업데이트 확인**: 실행 시 자동으로 최신 버전을 확인하고 필요한 경우 업데이트를 진행합니다. 잠시 기다려 주세요. 
   *(처음 실행 시에는 컴퓨터 환경에 따라 필수 프로그램(Node.js 등)을 자동으로 설치할 수 있습니다. 1~2분 정도 기다려 주세요.)*

## 🔑 최초 로그인 방법 (처음 한 번만)

1. 프로그램을 실행하면 검은색 창(터미널)과 함께 **인터넷 브라우저 창**이 자동으로 열립니다.
2. 해당 창에서 본인의 **키즈노트 아이디와 비밀번호로 로그인**해 주세요.
3. 로그인이 완료되면 브라우저 창을 닫지 말고 잠시 기다려 주세요. 프로그램이 로그인 정보를 인식하면 자동으로 창이 닫히고 백업이 시작됩니다.

## 📂 백업된 파일 확인

- 백업된 사진과 영상은 원본 화질 그대로 **`알림장`** 폴더와 **`앨범`** 폴더에 날짜별로 저장됩니다.
- 본문 내용과 선생님이 달아주신 댓글은 **`내용.txt`** 파일에 함께 저장됩니다.

## 💡 주의 사항

- **인터넷 연결**: 백업 중에는 인터넷이 연결되어 있어야 합니다.
- **창 닫지 않기**: 백업이 진행 중일 때는 검은색 창(터미널)을 닫지 마세요. "백업 완료" 메시지가 나오면 창을 닫으셔도 됩니다.
- **저장 경로 변경**: 기본적으로 프로그램이 있는 폴더에 저장됩니다. 만약 다른 곳(외장 하드 등)에 저장하고 싶다면 관리자에게 문의해 주세요.
